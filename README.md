# Java-Button-Counter

## Description
This is a simple Java application that demonstrates:

- Object-Oriented Programming (OOP)
- Event handling
- Button click counter logic
- Basic testing

## Files Included
- ButtonCounter.java
- ButtonCounterTester.java

## How It Works
Each time the button is clicked, the counter increments and displays the updated value.
